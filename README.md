# PalindromeCheckerApp
// Running Procedure:
Compile and run the Java program, enter a string through the console, and the program processes the input and displays whether it is a palindrome or not.
// Flow of Project:
Start → Accept user input → Process the string (format/clean if needed) → Reverse the string → Compare with original → Display result → End.
// Objective:
To develop a console-based Java application that checks whether a given string is a palindrome using basic programming and data structure concepts.
// Topics to be Covered:
Java basics, string manipulation, loops, conditional statements, user input using Scanner, and basic algorithm logic.
// Use Case:
A user enters a word or phrase, and the application checks and informs whether the entered text is a palindrome.
